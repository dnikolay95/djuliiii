# Этап 5. React UI (таблицы и детали) (done)

## Цель
Сделать удобный UI для просмотра данных.

## Задачи
- Экран Users: таблица + кнопка "Открыть детали".
- Детали пользователя: список поздравлений (время, текст) и сообщений (время, тип, текст/пометка).
- Экран Greetings: таблица, фильтр по пользователю.
- Экран Messages: таблица, фильтры по пользователю и типу.
- Сортировка/фильтры минимально достаточные.

## Результат проверки
- 3 экрана работают, данные читаются без realtime.
- Детали пользователя открываются и показывают его логи.
