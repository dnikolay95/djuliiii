# Этап 2. Логирование в боте без вебки (done)

## Цель
Записывать активность бота в БД, сохраняя текущую логику поздравлений.

## Задачи
- /start: upsert user.
- Любые входящие сообщения: upsert user + запись в messages_log (тип, текст, received_at).
- Отправка поздравления (callback get_greeting и /greet): upsert user + запись в greetings_log (user, текст, sent_at).
- Логика поздравлений как раньше (15 текстов, без дублирования кода).

## Результат проверки
- После /start и сообщений в БД есть записи users, messages_log.
- После поздравлений в БД есть записи greetings_log.
